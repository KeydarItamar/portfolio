Project for Hackaton Mobile World Congress 2024


As part of a team project for the MWC 2024 hackathon, we tackled the development of an application designed for pet geolocation. My primary role revolved around backend development, where I utilized Python to orchestrate API calls and construct a robust RESTful API to drive our platform. Additionally, I spearheaded the implementation of artificial intelligence scripts to augment the application's capabilities. While I focused on backend tasks, another team member assumed responsibility for frontend development.


Technologies i used:


  • Backend: Python
  • Api rest to provide endpoints: Fast Api
  • Framework of Artificial intelligence: Langchain
  • Large language model: Cohere(testing version)
  • Source code management: GitHub
